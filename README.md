# Toggle Button Component

A simple, modern toggle button built with **React** and **Tailwind CSS**. Lightweight, customizable, and accessible.

## Features
- Lightweight and easy to integrate  
- Smooth transition animations  
- Customizable via Tailwind CSS utilities  
- Accessible (`aria-pressed`)

## Requirements
- React  
- Tailwind CSS

## Customization
- Change colors, sizes, and transitions with Tailwind classes  
- Extract into a reusable component and expose `checked` / `onChange` props for control

## Accessibility
- Uses a semantic `<button>` and `aria-pressed` for assistive technologies  
- Keyboard-operable (Enter / Space)
